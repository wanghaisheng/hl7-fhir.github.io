1.14.4.1.21 v3 Code System Charset

This code system (http://hl7.org/fhir/v3/Charset) is defined as part of HL7 v3.

Formal value Set definition (identifier http://hl7.org/fhir/v3/vs/Charset): XML or JSON

Release Date: 2013-12-07

1.14.4.1.21.1 Description

LevelCodeDisplayDefinition
