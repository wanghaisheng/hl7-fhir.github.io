1.14.4.1.1052 v3 Code System ActSite

This code system (http://hl7.org/fhir/v3/ActSite) is defined as part of HL7 v3.

Formal value Set definition (identifier http://hl7.org/fhir/v3/vs/ActSite): XML or JSON

Release Date: 2013-12-07

1.14.4.1.1052.1 Description

An anatomical location on an organism which can be the focus of an act.

LevelCodeDisplayDefinition
1(_HumanActSite)  An anatomical location on a human which can be the focus of an act.
