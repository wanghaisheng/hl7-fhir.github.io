5.17.6 Resource Slot - Formal Definitions

Formal definitions for the elements in the Slot resource.

Slot
Definition(informative) A slot of time on a schedule that may be available for booking appointments.
Control1..1
Slot.identifier
DefinitionExternal Ids for this item.
Control0..*
TypeIdentifier
Slot.type
DefinitionThe type of appointments that can be booked into this slot (ideally this would be an identifiable service - which is at a location, rather than the location itself). If provided then this overrides the value provided on the availability resource.
Control0..1
TypeCodeableConcept
Slot.availability
DefinitionThe availability resource that this slot defines an interval of status information.
Control1..1
TypeResource(Availability)
Slot.freeBusyType
DefinitionBUSY | FREE | BUSY-UNAVAILABLE | BUSY-TENTATIVE.
